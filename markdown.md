# Text emphasis
* 굵게 -bold
    * `**` | `__`
    * JAVA
    * **JAVA**
    * __JAVA__

* 기울임 - italic
  * `*` | `_`
  * Python
  * *Python*
  * _Python*
  
* 밑줄 - underline
  * `<U>` `</U>` 사이에 작성
  * Spring
  * <U>Spring</U>

* 취소선 - strikethrough
  * `~~`로 감싸기
  * jQuery
  * ~~jQuery~~
